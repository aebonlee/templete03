## 1. 목표
신규 학습사이트를 빠르게 시작할 수 있는 **재사용 템플릿**을 구축한다.
`아트디렉션_템플릿.md` 스펙(A~I)을 디자인 기준선으로 삼아, 프로젝트마다
콘텐츠 1파일 + 팔레트 1파일만 교체하면 동작하도록 설계.

## 3. 아트디렉션 (다크 블루 5색)
| 역할 | 이름 | HEX | 비율 |
|---|---|---|---|
| Paper | 푸른 종이 | `#F4F6FB` | 60% |
| Ink | 자정의 남색 | `#14213D` | 30% |
| Pigment | 학술 블루 | `#1D4E89` | 8% |
| Accent | 형광 호박 | `#E8A33D` | 2% |
| Wash | 블루 워시 | `#DCE6F2` | — |

- 타이포: Noto Serif KR(제목) / Pretendard(본문) / JetBrains Mono(수치)
- **시그니처**: 형광펜이 좌→우로 그어지는 밑줄 연출(`Highlighter`). 섹션을 지날수록 형광 자국이 쌓인다.

## 4. 구현 내역
- **페이지**: 홈(히어로+가치3+커리큘럼+가격) · 커리큘럼 · 레슨(잠금/이전·다음) · 로그인
- **컴포넌트**: Header · Footer · Section · Reveal(스크롤 리빌) · Highlighter(시그니처) · Grain(종이결)
- **인증**: `AuthContext` + Supabase OAuth(Google/Kakao), env 미설정 시 graceful fallback
- **DB**: `supabase/schema.sql` — `tpl_profiles / tpl_progress / tpl_notes / tpl_orders` + RLS + 가입 트리거. JS는 `t('progress')` 헬퍼로 접두사 처리
- **설정 주도**: `src/config/site.js`(콘텐츠) + `src/styles/tokens.css`(팔레트)만 교체
- **OG**: index.html 메타태그 + `scripts/generate-og.mjs`(sharp로 다크블루 1200×630 PNG 생성)
- **배포**: `.github/workflows/deploy.yml` (main push → build → Pages, SPA 404 fallback + CNAME)

## 5. 보안 처리
- 비밀 키는 `.env.local`(gitignore)에만 보관, `.env.example`로 형태만 커밋
- `SUPABASE_ACCESS_TOKEN`(sbp_)은 CLI 전용 — **VITE_ 접두사 금지**(클라이언트 비노출)
